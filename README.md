# UWP-GrovePi-Test-Bench
A UWP app to test sensors on a GrovePi board hooked to a Windows 10 Raspbery Pi.
ARM only.

Currently supported:
* Button
* LED
* Light Sensor
* PIR Motion Sensor
* Rotary Angle Sensor (a.k.a knob)
* Temperature Sensor
* Vibration Motor

Working on:
* Infrared Emitter
* Infrared Receiver

Having a problem with:
* LED Bar
