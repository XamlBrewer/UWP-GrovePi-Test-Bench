# UWP-GrovePi-Test-Bench
A UWP app to test sensors and actuators on a GrovePi board hooked to a Windows 10 Raspbery Pi. So it's ARM only.

Currently supported:
* Button
* LED
* Light Sensor
* PIR Motion Sensor
* Rotary Angle Sensor
* Temperature Sensor
* Vibration Motor

Doing research on:
* Infrared Emitter
* Infrared Receiver

Having a problem with:
* LED Bar

Note: runs best (= more stable) in Release mode.
